## Shared Pages

페이지 테이블에 있는 수는 실제 frame번호들

페이지 테이블을 같은걸 지정해놓으면 페이지 공유 가능.

물리적으로 같은 주소를 가리킴

32bit주소 

주소는 0 ~ 2^32-1까지의 수로 표현 가능

page(p) 는 0 ~ 2^20-1

페이지 테이블은 frame번호를 기록

* frame(f) 번호 : 20bit

page table은 250MB인데 페이지는 4KB

* 페이지 테이블이 여러 개의 페이지에 저장된다는걸 의미
